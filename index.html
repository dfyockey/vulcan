vulcan
3d chess playing program

Huh?

    vulcan lets you play against the computer a chess variant inspired by a certain well-known science fiction TV series.

    See the README file for more information, or this for some assorted notes for the tech-oriented.

    vulcan is written in C, and was successfully compiled and tested on Linux. In order to compile it, you'll need lex and yacc, and OpenGL libraries and header files. As of version 0.5, you'll also need libpng.
